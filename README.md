# Fullstack Task Management System

A comprehensive task management application built with a modern tech stack. This project demonstrates the integration between a robust C# backend and a dynamic JavaScript frontend.

## Features
- **Task Creation & Management:** Create, update, and delete tasks efficiently.
- **RESTful API:** Developed using C# (.NET) to handle data logic and requests.
- **Responsive UI:** Built with JavaScript, CSS, and HTML for a smooth user experience.
- **Docker Support:** Includes a Dockerfile for easy containerization and deployment.

## Tech Stack
- **Backend:** C# (.NET Core API)
- **Frontend:** React.js (Vite), HTML5, CSS3
- **Tools:** Docker, Axios (for API calls)
